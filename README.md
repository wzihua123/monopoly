一个模拟职业生涯发展的微信小程序游戏，通过掷骰子在双层地图上移动，体验不同的职业发展路径，努力实现金钱、幸福和名誉的平衡发展。

## 游戏流程详解

### 1. 新手引导
- 游戏开始时自动弹出游戏规则说明
- 介绍基本玩法和胜利条件
- 解释各类格子和通道的作用

### 2. 难度选择
三个难度等级供选择：
- 初等难度：总目标分 60分
- 中等难度：总目标分 80分
- 高等难度：总目标分 100分

### 3. 目标值设定
- 玩家需设定三项属性的目标值：
  - 金币积分（100金币 = 1金币积分）
  - 幸福值
  - 名誉值
- 三项属性目标值之和必须等于所选难度的总分
- 示例（初等难度）：
  - 金币积分：50分（需要5000金币）
  - 幸福值：5分
  - 名誉值：5分
  - 总计：60分

### 4. 起始资金抽奖
随机获得以下之一：
- 500金币
- 1000金币
- 1500金币

### 5. 游戏进行
1. 投骰子决定移动步数
2. 根据落点获得相应奖励或惩罚
3. 可选择进入职业通道发展
4. 使用经验卡和机会卡
5. 实时显示当前属性值与目标的差距

### 6. 胜利条件
1. 三项属性都达到设定的目标值
2. 三项属性之和达到难度要求的总分
3. 记录不同难度的通关次数

### 游戏数据记录
系统会记录玩家：
- 各难度通关次数
- 最快通关回合数
- 获得的最高属性值
- 解锁的职业通道数量

### 游戏建议
1. 合理分配目标值，避免单项过高
2. 根据起始资金选择合适的发展路线
3. 优先选择与目标值匹配的职业通道
4. 灵活运用经验卡和机会卡
5. 注意现金流管理，预留进入职业通道的费用
